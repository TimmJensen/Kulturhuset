# Installation & Kørsel
Step-by-step.
Download og open mappen i en terminal.

1. Installér modules.
```
npm install
```

2. Kør server
```
npm run
```

3. Åben en browser og gå ind på [Localhost:3000](http://localhost:3000/)

## Modules: 
- Express
- EJS
- HTTP-Error
- Cookie-parser
- Morgan
- Nodemon
- Mongodb

## Routes
GET:
- / <-- Index
- /admin <-- Admin Kontrol Panel

POST: 
- /opdater-titel <-- Opdatere titel i DB
